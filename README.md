# Межевой план (Megevoy Plan)

**Межевой план** — это кроссплатформенное десктопное приложение с открытым исходным кодом для кадастровых инженеров, позволяющее создавать, редактировать и экспортировать межевые планы в формате XML, полностью соответствующем официальной схеме Росреестра **MP_v09.xsd**.

Проект разработан в рамках выпускной квалификационной работы и размещён на GitHub: [https://github.com/abramovsergeyn/megevoy-plan](https://github.com/abramovsergeyn/megevoy-plan)

---

## Основные возможности

- **Управление проектами** – создание, открытие, сохранение проектов межевания.
- **Работа с участками** – добавление, редактирование, удаление земельных участков (образуемых, изменяемых, смежных).
- **Ввод координат** – ручной ввод характерных точек, импорт из CSV и KML.
- **Геометрические проверки** – автоматическое вычисление площади, контроль замкнутости контура, обнаружение самопересечений и пересечений между участками.
- **Адрес в структурированном виде** – заполнение адреса по элементам ФИАС (регион, район, населённый пункт, улица, дом и т.д.).
- **Справочники** – предустановленные классификаторы (категории земель, виды разрешённого использования, регионы, методы определения координат, типы обременений) с возможностью их расширения.
- **Экспорт в XML** – генерация XML-документа межевого плана, полностью соответствующего схеме MP_v09.xsd.
- **Валидация XML** – проверка сгенерированного файла по XSD-схеме.
- **Логирование** – запись всех действий и ошибок в лог-файл.
- **Тестирование** – модульные, интеграционные и функциональные тесты (с возможностью генерации случайных данных).

> **Примечание:** Формирование PDF-чертежей и ZIP-архива пакета будет реализовано в следующих версиях. В текущей версии подготовлена структура для их добавления.

## Технологический стек

- **Язык**: Python 3.11
- **GUI**: PySide6 (Qt for Python)
- **ORM**: SQLAlchemy 2.0 + SQLite
- **XML**: lxml (генерация, валидация по XSD)
- **Геометрия**: Shapely, pyproj
- **Тестирование**: pytest, pytest-qt

## Установка

### Требования
- Python 3.11 или выше
- pip (менеджер пакетов)

### Шаги

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/abramovsergeyn/megevoy-plan.git
   cd megevoy-plan

2. (Рекомендуется) Создайте виртуальное окружение:
bash
python -m venv venv
source venv/bin/activate      # Linux/Mac
venv\Scripts\activate         # Windows

3. Установите зависимости:
bash
pip install -r requirements.txt
________________________________________
Запуск
Запустите приложение командой:
bash
python -m src.main
При первом запуске автоматически создаётся база данных megevoy.db и заполняются справочники из JSON-файлов в папке resources/dictionaries/.

Использование
Главное окно
•	Меню «Файл»: создание/открытие проекта, импорт координат (CSV, KML), экспорт (XML межевого плана, координаты в CSV).
•	Меню «Правка»: добавление/редактирование/удаление участков, управление справочниками инженеров и заказчиков.
•	Панель инструментов: быстрый доступ к операциям с участками.
•	Таблица участков: отображает список участков текущего проекта.
Создание нового проекта
1.	Выберите «Файл → Новый проект».
2.	Введите название, выберите кадастрового инженера и заказчиков (можно создать новых через кнопки «Управление…»).
3.	Нажмите «ОК».
Добавление участка
1.	Выберите проект, нажмите «Добавить участок».
2.	В открывшемся диалоге заполните:
o	Основное: кадастровый номер (опционально), статус, кадастровый квартал, категория земель, вид разрешённого использования.
o	Координаты: добавьте точки с координатами (X, Y). Для проверки геометрии нажмите «Проверить геометрию».
o	Адрес: укажите структурированный адрес (регион обязателен).
3.	Нажмите «ОК».
Импорт координат
•	CSV: выберите «Файл → Импорт → Координаты из CSV». Формат: номер, X, Y (с разделителем-запятой). Можно пропустить заголовок.
•	KML: выберите «Файл → Импорт → Границы из KML». Извлекаются координаты первого полигона.
Экспорт межевого плана
Выберите «Файл → Экспорт → Межевой план в XML». Укажите имя файла. Будет создан XML, соответствующий схеме MP_v09.xsd.
Проверка геометрии
Выделите участок в таблице и нажмите «Проверить геометрию» на панели инструментов. Отобразится информация о замкнутости, самопересечениях и площади.
Управление справочниками
Через меню «Правка» → «Кадастровые инженеры» или «Заказчики» можно добавлять, редактировать и удалять записи.

Тестирование
Проект покрыт тестами трёх уровней:
•	Модульные тесты – проверка отдельных функций (геометрические вычисления, генерация XML и т.д.).
•	Интеграционные тесты – взаимодействие модулей (например, импорт → сохранение → экспорт → валидация).
•	Функциональные тесты – эмуляция действий пользователя в GUI (с использованием pytest-qt).
Запуск тестов
bash
pytest tests/ -v
Для проверки покрытия:
bash
pytest --cov=src tests/
При тестировании можно использовать как ручной ввод данных, так и автоматическую генерацию случайных, но валидных данных (реализовано в фикстурах).
